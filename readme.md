## 📖 Описание на проекта
**CADS** е уеб-базирана аналитична система (Decision Support System), която използва алгоритми за дълбоко машинно обучение (Deep Learning), за да анализира финансови времеви редове. Системата извлича данни в реално време, филтрира пазарния шум и предоставя на потребителя прогноза за краткосрочния тренд. 

Най-важната функционалност на системата е **Откриването на аномалии (Anomaly Detection)**. Чрез изчисляване на статистическото отклонение между прогнозата на изкуствения интелект и реалната пазарна стойност, CADS алармира потребителя за наличието на екстремна волатилност и пазарни шокове.

## ✨ Основни функционалности
* **Интеграция в реално време:** Автоматично изтегляне на OHLCV данни чрез `yfinance` API.
* **Изкуствен интелект:** Прогнозиране на ценови движения чрез Stacked LSTM невронна мрежа с Attention механизъм.
* **Управление на риска:** Визуална индикация (Аларми) при засичане на пазарна аномалия (отклонение > 5%).
* **Потребителски профили:** Защитена автентикация (Flask-Login) и съхранение на лична история на направените анализи.
* **MLOps Административен панел:** Възможност за фоново пре-обучение (Background Retraining) на модела директно през уеб интерфейса с най-актуалните пазарни данни.

## 🏗️ Архитектура на системата
Проектът следва **Трислойна архитектура (MVC)** и шаблона **Application Factory**:
* **Presentation Layer:** HTML5, CSS3 (Bootstrap 5), JavaScript (Chart.js).
* **Business Logic Layer:** Python (Flask), TensorFlow/Keras за ML инференция.
* **Data Access Layer:** SQLite чрез SQLAlchemy ORM.
