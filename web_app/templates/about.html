За Проекта CADS

Crypto Anomaly Detection System (CADS) е дипломен проект, разработен с цел изследване на възможностите на изкуствения интелект във финансовия сектор.

Технологичен Стек:

  • Machine Learning: TensorFlow, Keras, Scikit-Learn
  • Мрежова архитектура: Stacked LSTM (Long Short-Term Memory)
  • Бекенд: Python, Flask, SQLite
  • Източник на данни: Yahoo Finance API (Реално време)

Отказ от отговорност: Тази система е разработена с академична цел. Прогнозите не представляват финансов съвет.
